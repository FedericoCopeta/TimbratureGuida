1.0 Premessa

Questo documento descrive le funzionalità e le modalità di utilizzo del file excel creato per tenere traccia delle timbrature, dei buoni pasto e delle ferie/permessi richiesti e/o utilizzati. [cite: 3]

Il file è stato progettato per automatizzare tutti i calcoli necessari per monitorare le attività lavorative durante l'anno, richiedendo all'utente solo l'inserimento degli orari di entrata e uscita. [cite: 4, 5] Tutti i calcoli (esuberi, flessibilità, ferie, permessi, buoni pasto) sono automatizzati secondo le specifiche dell’Università del Piemonte Orientale. [cite: 6]

2.0 Installazione e primo utilizzo

Per il corretto funzionamento, il file utilizza formule e macro VBA. [cite: 27] È quindi necessario assicurarsi che le macro siano abilitate. [cite: 27]

Abilitare le Macro

  1. Cliccare con il tasto destro sul file e selezionare Proprietà. [cite: 29]
  2. Nella scheda "Generale", se presente in basso la sezione "Sicurezza", spuntare la casella "Annulla blocco". [cite: 30, 31]
  3. Aprire il file e, se compare un avviso di sicurezza, cliccare su Abilita contenuto. [cite: 32]

Questa operazione va eseguita solo la prima volta che si utilizza il file sulla postazione di lavoro o ogni volta che il file viene copiato o rinominato. [cite: 33]

Foglio Riepilogo

Il foglio "Riepilogo" è un cruscotto che riassume i dati di tutti i mesi e contiene i pulsanti per gestire i fogli di lavoro mensili. [cite: 39]

  • Importa Timbrature: Permette di importare timbrature da un altro file Excel per evitare di reinserirle manualmente. [cite: 42] ATTENZIONE: I dati nel file corrente verranno sovrascritti. [cite: 44]
  • Data Odierna: Porta il cursore alla data odierna. [cite: 54]
  • Crea Foglio Mese: Crea un nuovo foglio per il mese specificato. [cite: 61]
  • Cancella Foglio Mese: Elimina il foglio di un mese, aggiornando le formule. [cite: 67]
  • Crea Nuovo Anno: Crea un nuovo file per l'anno specificato. [cite: 70]

3.0 Utilizzo del file

È sufficiente compilare le celle di entrata e uscita nel formato hh:mm. [cite: 79] Il file calcola automaticamente le ore lavorate, l'eventuale esubero (FLEP) o debito (FLEN) e assegna il buono pasto se si superano le 6:12 ore di lavoro. [cite: 85, 86, 88]

MOPER

L'orario di lavoro ha un'entrata flessibile tra le 8:00 e le 9:00. [cite: 90] Entrando dopo le 9:00, il ritardo viene segnato in rosso e aggiunto alla colonna MOPER. [cite: 91]

Ferie/permessi

Per inserire ferie o permessi, usare i seguenti codici nella prima cella di entrata (Entrata):

  • Ferie: Inserire F. [cite: 99]
  • Permessi (P18H): Inserire P18H. [cite: 100]
  • Festività: Inserire Fest. [cite: 104]
  • Malattia: Inserire M. [cite: 107]
  • Recupero ore: Inserire R. [cite: 110]
