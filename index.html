Guida Timbrature

Benvenuto nella guida ufficiale.

Questo documento descrive le funzionalità e le modalità di utilizzo del file excel creato per tenere traccia delle timbrature, dei buoni pasto e delle ferie/permessi richiesti e/o utilizzati.

Il file è stato progettato per automatizzare tutti i calcoli necessari per monitorare le attività lavorative durante l'anno, richiedendo all'utente solo l'inserimento degli orari di entrata e uscita. Tutti i calcoli (esuberi, flessibilità, ferie, permessi, buoni pasto) sono automatizzati secondo le specifiche dell’Università del Piemonte Orientale.

Nota: Usa il menu sopra o la barra di ricerca per navigare tra gli argomenti.

Per iniziare

Questa sezione contiene le informazioni preliminari.

Prerequisiti

Per utilizzare questo strumento è necessario disporre di Microsoft Excel (versione 2016 o successiva consigliata).

Installazione e primo utilizzo

Per il corretto funzionamento, il file utilizza formule e macro VBA. È quindi necessario assicurarsi che le macro siano abilitate.

Abilitare le Macro

  1. Cliccare con il tasto destro sul file scaricato e selezionare Proprietà.
  2. Nella scheda "Generale", in basso nella sezione "Sicurezza", spuntare la casella "Annulla blocco" e cliccare OK.
  3. Aprire il file Excel. Se compare un avviso di sicurezza giallo in alto, cliccare su Abilita contenuto.

Questa operazione va eseguita solo la prima volta che si utilizza il file sulla postazione di lavoro.

Riepilogo e Dashboard

Foglio Riepilogo

Il foglio "Riepilogo" è un cruscotto che riassume i dati di tutti i mesi. Da questo foglio di lavoro è possibile avere sotto controllo i totali di tutti i contatori delle ore lavorative (Flessibilità positiva/Negativa, Ferie, buoni pasto, etc.).

Oltre alla consultazione dei totali, questo foglio permette la gestione dei fogli mensili tramite appositi pulsanti (Creazione mese, Cancellazione mese, Nuovo anno).

Accesso e Navigazione

È possibile accedere al Riepilogo selezionando la linguetta in basso o cliccando sul pulsante Riepilogo nel menu TIMBRATURE.

Funzionalità del pulsante Riepilogo:
Il pulsante nel menu TIMBRATURE ha una doppia funzione: cliccandolo si viene portati al foglio Riepilogo; cliccandolo nuovamente, si viene riportati automaticamente al mese su cui si stava lavorando prima del click.

Dashboard

La Dashboard offre una rappresentazione visuale dell'andamento annuale tramite grafici intuitivi. È accessibile cliccando sul pulsante Dashboard nel menu TIMBRATURE.

Navigazione e Chiusura Automatica:
Il foglio Dashboard ha un comportamento dinamico per mantenere l'interfaccia pulita:
  • Chiusura automatica: Se si clicca su un qualsiasi altro foglio di lavoro (es. Gennaio), la Dashboard viene nascosta automaticamente.
  • Tasto "Torna indietro": Cliccando nuovamente sul pulsante Dashboard nel menu, il foglio verrà nascosto e si verrà riportati al mese in cui si stava lavorando.

Utilizzo del file

È sufficiente compilare le celle di entrata e uscita nel formato hh:mm.

Il file calcola automaticamente le ore lavorate, l'eventuale esubero (FLEP) o debito (FLEN) e assegna il buono pasto se si superano le 6 ore e 12 minuti di lavoro.

MOPER (Motivi Personali)

L'orario di lavoro ha un'entrata flessibile tra le 8:00 e le 9:00. Entrando dopo le 9:00, il ritardo viene segnato in rosso e aggiunto alla colonna MOPER.
