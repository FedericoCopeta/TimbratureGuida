Benvenuto

Novità: Gestisci tutto dal nuovo menu "TIMBRATURE" in alto su Excel! Scopri le nuove funzioni nella sezione dedicata.

Il file Timbrature UPO è uno strumento automatizzato progettato per semplificare la gestione mensile dell'orario lavorativo. Ti permette di registrare entrate e uscite ottenendo in tempo reale il calcolo di flessibilità, buoni pasto e residuo ferie, in conformità con le normative dell'Ateneo.

In questa guida troverai tutte le istruzioni per configurare il file e risolvere eventuali dubbi comuni.

Argomenti consigliati

  • Scopri il nuovo Menu in alto (Ribbon)
  • Nuovi Codici Speciali (VISMED, L104, ecc.)
  • Come gestire i ritardi (MOPER)

Installazione e primo utilizzo

Il file utilizza automazioni avanzate (VBA). Per questo motivo, Windows e Excel potrebbero richiedere un'autorizzazione di sicurezza al primo avvio.

1. Sblocco del file (Windows)

Prima di aprire il file per la prima volta:

  1. Tasto destro sul file scaricato > Proprietà.
  2. In basso a destra, spunta la casella "Annulla blocco" e premi OK.

2. Abilitazione contenuto (Excel)

All'apertura del file, noterai una barra gialla sotto il menu.

  1. Clicca sul pulsante Abilita contenuto per attivare i calcoli automatici.

Il Menu TIMBRATURE (Barra multifunzione)

Tutte le operazioni principali sono ora gestibili dal pannello di controllo unificato situato nella parte superiore di Excel.

🛠️ Tools (Gestione File)

  • Crea Nuovo Anno: Genera un nuovo file per l'anno successivo mantenendo intatte le impostazioni.
  • Crea Nuovo Mese: Aggiunge il mese in ordine cronologico.
    Nota: L'inserimento o l'eliminazione manuale dei fogli tramite tab Excel è ora bloccata per prevenire danni alle formule. Usa sempre questo pulsante!
  • Cancella Mese: Rimuove in modo sicuro l'ultimo mese, ripristinando il sistema.

📍 Posizione (Navigazione)

  • Giorno Precedente: Ti riporta istantaneamente all'ultima riga lavorata del giorno prima.
  • Vai a Oggi: Cerca il foglio corrente e seleziona la riga di oggi in automatico.

⏱️ Utility (Calcoli e Dati)

  • Calcola Uscita: Calcola l'orario ideale di uscita aggiungendo 6 ore e 12 minuti.
  • Calcolatrice Completa: Apre un modulo per simulare timbrature, pause ed esuberi.
  • Aggiorna Festività: Propaga eventuali modifiche al calendario festività a tutti i fogli.
  • Esporta / Importa Dati: Crea e ripristina backup (CSV) delle tue ore. Totalmente compatibile con macOS.

📊 Report (Statistiche)

  • Dashboard: Apre i grafici annuali. Clicca la freccina sotto l'icona per Esportare in PDF.
  • Riepilogo: Vai rapidamente alla matrice dati annuale.

⚙️ Manutenzione (Avanzate)

Area protetta da password amministratore per sbloccare la struttura, forzare l'aggiornamento delle formule o mostrare i fogli di calcolo nascosti.

Utilizzo del file

Nei fogli mensili devi inserire solo:

  • Orario di Entrata (es. 08:05)
  • Orario di Uscita (es. 14:00)

Il calcolo di FLEP (Esubero), FLEN (Debito) e Buoni Pasto è immediato.

Gestione Ritardi (MOPER)

La fascia di flessibilità è 08:00 - 09:00. Se timbri dopo le 09:00, il sistema segnerà il ritardo in rosso e lo conteggerà automaticamente nella colonna MOPER (Motivi Personali).

Assenze e Permessi speciali

Oltre a Ferie e Malattia, l'algoritmo riconosce ora una vasta gamma di permessi inserendoli direttamente nella colonna Entrata (es: L104, LUTTO). Consulta la sezione Legenda per l'elenco completo.
